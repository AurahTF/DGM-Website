Font Forge

FontForge is a free and open source font editor brought to you by a community of fellow type lovers. You can donate to support the project financially.
Download Github

Get help, ask a question on the mailing list if you're stuck and the documentation and a websearch didn't provide any answers.

Get involved, anyone can help! You don't have to be a programmer. If you want to help but don't know where then join the developer list and introduce yourself.

Get started, we recommend that you start by reading Design With FontForge before moving on to the documentation on this page.
